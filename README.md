# FPGA Guessing Word Game (Digital Logic Project)

## Overview
This project implements a **Guessing the Word game** using **Verilog HDL** and digital logic circuits.  
The system runs on an FPGA and allows a user to guess a randomly generated three-letter word using hardware switches.

The game logic compares user inputs with the stored word and displays the results on **7-segment displays**.

This project was developed as part of the **Digital Logic Laboratory course** at the **University of Jordan**.

---

## Game Description

The player must guess a **3-letter word** generated by the system.

Allowed letters:

A B C F H I L O P S T U

The player enters letters using **4-bit switch inputs**.

The system:

• checks whether the letter exists in the word  
• updates the guess display  
• decreases the number of trials if the guess is wrong  

The player has **5 trials** to guess the word.

---

## Inputs

- Start Switch
- New Word Switch
- Enter Switch
- 4-bit Letter Input Switches

---

## Outputs

- LED1 → Game started
- LED2 → Player wins
- LED3 → Player loses
- Letter 7-Segment Display
- Guess 7-Segment Display
- Trials 7-Segment Display

---

## System Architecture

The system is composed of several Verilog modules:

### Word Register
Stores the randomly generated 3-letter word.

### Letter Register
Stores the user's guessed letter.

### Comparator
Compares the guessed letter with the word letters.

### Trial Register
Tracks remaining attempts.

### 7-Segment Decoder
Displays letters and remaining trials.
